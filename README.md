# MediaBridge App

এই রিপোজিটরি MediaBridge অ্যাপের সোর্সকোড রাখে। নিচে দ্রুত শুরু করার নির্দেশনা, প্রয়োজনীয়তা, এবং অবদান রাখার নিয়ম দেওয়া আছে।

## সারসংক্ষেপ
MediaBridge একটি মক্স/অ্যাপ/প্রজেক্ট (এখানে প্রকল্পের সংক্ষিপ্ত বিবরণ দিন)।

## বৈশিষ্ট্যসমূহ
- মিডিয়া ফাইল শেয়ার/স্ট্রিম (আপনার প্রকল্প অনুযায়ী আপডেট করুন)
- মিডিয়া প্রসেসিং
- ইত্যাদি

## প্রয়োজনীয়তা
- Java 11+ / OpenJDK
- Android Studio (সংস্করণ অনুযায়ী)
- Android SDK এবং প্ল্যাটফর্ম টুলস
- Gradle

## লোকালি বিল্ড করা
1. রিপো ক্লোন করুন:

   git clone https://github.com/elephantcos-cloud/mediabridge-app.git
   cd mediabridge-app

2. Android Studio-এ ওপেন করে Build → Make Project চালান।

3. অথবা টার্মিনাল থেকে (Unix/macOS/Windows Git Bash):

   ./gradlew assembleDebug

বিল্ট হওয়া APK `app/build/outputs/apk/` এ পাবেন (প্রজেক্ট কনফিগারেশনের ওপর নির্ভর করে)।
